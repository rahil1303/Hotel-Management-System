# Hotel Management System

This project involves the development of software to automate various hotel functionalities, including room booking and management. The software was developed using NetBeans IDE and MySQL for database management.

## Project Structure

- **Source Code**: Contains the Java source files for the hotel management system.
- **Database Scripts**: Includes SQL scripts for creating and populating the MySQL database used in the project.
- **Documentation**: Any additional documentation or user manuals related to the project.

## Key Features

- **Room Booking**: Allows customers to book rooms based on availability and preferences.
- **Room Management**: Enables hotel staff to manage room availability, check-ins, and check-outs.
- **Customer Management**: Tracks customer details and booking history.
- **Billing System**: Automatically generates bills based on room rates and additional services.
- **User Authentication**: Provides secure login for different user roles, including admin and hotel staff.

## Installation

To run this project, you need to have the following installed:

- **Java JDK**: Version 8 or above.
- **NetBeans IDE**: Preferably the version used during development, but later versions should work as well.
- **MySQL**: For managing the project database.
